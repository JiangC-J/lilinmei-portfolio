# GitHub Pages Deploy

Use this skill when preparing deployment.

- Configure GitHub Pages deployment.
- If using `gh-pages`, add scripts to `package.json`.
- Document these commands in `README.md`:
  - `npm install`
  - `npm run dev`
  - `npm run build`
  - `npm run deploy`
- If deployment path is not root, set Vite `base` correctly.
