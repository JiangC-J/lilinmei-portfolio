# Project Case Study

Use this skill when creating or editing project detail pages.

Each detail page must include:

- Large project cover.
- Project Info: project type, time, tools, and personal responsibilities.
- Background: short project context.
- Keywords: 3 to 5 visual keywords.
- Design System: colors, type, graphics, logo, icon, UI, or packaging rules.
- Process: sketches, derivation, components, and visual process.
- Final Display: mockups, applications, page groups, or scene summaries.
- Summary: project highlights and learning.

Do not only show images. Add short, useful explanatory text.
