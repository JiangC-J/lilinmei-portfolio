# Visual System

Use this skill when changing layout, CSS, color, or visual tone.

- Use `#FEF7EA` or `#FEFBF6` for primary backgrounds.
- Use `#3F2A1D` for main text.
- Use `#F6B93B` and `#FF9A2E` as accents.
- Use `#8C6A4A` as secondary text.
- Allow subtle paper texture, rounded cards, hand-drawn lines, and small graphic details.
- Use clear system Chinese type for body text. Do not sacrifice readability for decoration.
- Make the website feel like a scrollable hand-made visual magazine.
