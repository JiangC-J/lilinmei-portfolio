# Portfolio IA

Use this skill when planning or revising the portfolio structure.

- Make the homepage include Hero, About, Selected Works, Design Workflow, Tools, and Contact.
- Make `/works` show all project cards.
- Make `/works/:slug` include project cover, project info, background, visual keywords, design process, final display, and summary.
- Do not stack long PDF images directly into pages. Split portfolio content into readable web modules.
- Keep navigation simple and useful for internship or job-review contexts.
