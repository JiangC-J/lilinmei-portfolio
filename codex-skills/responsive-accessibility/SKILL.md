# Responsive Accessibility

Use this skill when reviewing or changing layout and content.

- Use 2 to 3 columns for desktop project cards.
- Use 1 column for mobile project cards.
- Keep font sizes readable.
- Make buttons, links, and navigation clear.
- Add alt text to every meaningful image.
- Ensure color contrast is sufficient.
- Do not place text over complex images where readability suffers.
