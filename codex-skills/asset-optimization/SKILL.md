# Asset Optimization

Use this skill when adding, replacing, or documenting project images.

Create folders under `public/assets/projects/`:

- `tipsy-nest`
- `gugu-song`
- `pamper`
- `yujian`
- `cod-fish`
- `xiaonaoye`

Each project must have at least `cover.jpg`.

- Use placeholder cards if real images are missing.
- Document replacement locations in `README.md`.
- Use `loading="lazy"` for images.
- Compress large images before publishing.
